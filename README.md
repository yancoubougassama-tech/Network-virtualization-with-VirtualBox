# Network-virtualization-with-VirtualBox
Ce projet démontre la mise en place d'un environnement de virtualisation réseau en utilisant VirtualBox. Cet environnement réseau virtualisé est composé de plusieurs machines virtuelles (clients, serveurs, routeur virtuel). L’objectif est de reproduire un réseau local fonctionnel permettant de tester la connectivité, les échanges et la configuration réseau dans un environnement isolé.
## Objectifs du projet
- Comprendre les principes de la virtualisation et l’usage d’un hyperviseur de type 2
- Créer plusieurs machines virtuelles pour simuler un réseau complet
- Configurer les modes réseau VirtualBox (NAT, Bridge, Host‑Only, Internal)
- Tester la communication entre machines virtuelles
- Analyser le trafic réseau
- Deployer un routeur, un serveur DHCP, un serveur HTTP
- Mettre en plce un Firewell iptables
## Actions principales
- Création des VM dans VirtualBox
- Configuration des interfaces réseau virtuelles
- Installation des OS invités
- Mise en place de la topologie réseau
- Tests de connectivité entre VM
## Technologies
- VirtualBox
- Ubuntu Server
- Apache2
- isc-dhcp-server
- iptables
- Wireshark
